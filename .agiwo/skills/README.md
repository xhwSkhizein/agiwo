# agiwo built-in Skills

## working-with-files

> reference: [OthmanAdi/planning-with-files](https://github.com/OthmanAdi/planning-with-files)

This skill allows the agent to work with files.


## making-plan

> reference: [agiwo/making-plan](https://github.com/agiwo/making-plan)

Helps users make a "good" plan for a task.


## find-skills

> reference: [agiwo/find-skills](https://github.com/agiwo/find-skills)

Helps users discover and install agent skills when they ask questions like "how do I do X", "find a skill for X", "is there a skill that can...", or express interest in extending capabilities. This skill should be used when the user is looking for functionality that might exist as an installable skill.
